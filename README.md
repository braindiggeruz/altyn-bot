# Алтын Therapy Bot + Admin Panel

Telegram-бот с квизом, автопрогревом и маркетинговой админ-панелью для гипнотерапевта Алтын.

## Функции

### Telegram-бот (@altyntherapybot)
- Приветствие с красивой картинкой
- Квиз из 7 вопросов → 4 сценария (Спасатель, Страх, Контроль, Заморозка)
- Персональный результат с картинкой
- Автосерия прогрева (5 дней)
- Запись на бесплатную диагностику
- Уведомления владельцу о новых заявках

### Админ-панель
- Дашборд: статистика в реальном времени
- Пользователи: таблица с фильтрами
- Рассылки: по сегментам аудитории
- Воронка: конверсия на каждом этапе
- Аналитика: источники, сценарии, графики

## Стек
- Node.js + Express
- node-telegram-bot-api
- better-sqlite3
- node-cron

## Переменные окружения
- `BOT_TOKEN` — токен Telegram-бота
- `PORT` — порт для админ-панели (по умолчанию 3000)
- `OWNER_TELEGRAM_ID` — Telegram ID владельца для уведомлений
- `JWT_SECRET` — секрет для JWT-токенов
